Instalacion de Nodejs

debemos de tener instalado node en caso de que no puede descargarlo de link

nodejs

Comando para instalar Angular/CLI

Debemos abrir una Terminal (Mac/Linux) o cmd (windows) colocamos el siguiente comando

(windows) npm install -g @angular/cli

(Mac/Linux) sudo npm install -g @angular/cli

Validar instalacion

Para validar la instalacion colocar el comando en la terminal ng version (ng v)

con la cual nos mostrara la imagen de la version, sistema operativo en el cual esta instalada, asi mismo valida la version de nodejs.
